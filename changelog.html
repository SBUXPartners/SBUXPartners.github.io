Tippy Changelog

Version 1.0 - August 11, 2024

Initial Release

  • Basic tip calculation functionality
  • Add and remove barista entries
  • Mobile-responsive design
  • Uniform hourly tip rate calculation

Added

  • Dark mode toggle for improved visibility in low-light conditions
  • Automatic focus progression when pressing 'Enter' on input fields
  • New styling for the hourly tip rate display in the summary

Improved

  • Enhanced visual appeal of the "Tip Distribution Summary" section
  • Updated layout of summary items for better readability
  • Improved consistency with the overall modern UI design
  • Standardized "Delete" button appearance to match other buttons
  • Adjusted store information text color for better visibility in dark mode
  • Distinct visual separation between summary items
  • Enhanced visibility of summary details in dark mode

September 17, 2024

  • Enhanced hourly tip rate calculation and display:
    • Now showing full hourly tip rate with up to 5 decimal places
    • Displaying both full and rounded hourly tip rates in the summary
  • Improved tip calculation transparency:
    • Now calculating and displaying tips for each barista using both full and rounded hourly rates
    • Summary now shows both full and rounded tip amounts for each barista
  • Updated terminology:
    • Changed "Truncated Hourly Tip Rate" to "Rounded Hourly Tip Rate" for better clarity
  • Improved summary display:
    • Added clearer formatting for individual barista tip breakdowns
    • Enhanced readability of full vs. rounded tip calculations
